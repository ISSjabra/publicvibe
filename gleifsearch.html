How to Use This Tool

1. Search for a Company:

Type the name of a legal entity into the search bar and click "Search" or press Enter. The tool will find matching records from the GLEIF database.

2. Use a URL Parameter:

You can also search directly by adding a query to the URL. This is useful for sharing links to specific search results.

Example: .../your-file.html?query=Google

Legal Entity Search

Find Legal Entity Identifiers (LEI) using the GLEIF API.

Data provided by the Global Legal Entity Identifier Foundation (GLEIF).
